# 작업계획표

| 단계 | 작업 | 산출물 | 담당 파일 | 우선순위 |
|---|---|---|---|---|
| 1 | VS Code 프로젝트/가상환경 설정 | 실행 가능한 Streamlit 기본 앱 | `app.py`, `requirements.txt` | 높음 |
| 2 | 데이터 수집 함수 작성 | 시장, 금리, 거시, 유동성 CSV | `utils/*_data.py`, `data/raw/` | 높음 |
| 3 | 전처리 파이프라인 작성 | 주간화, 병합, lag 변수, 최종 데이터셋 | `utils/preprocessing.py`, `data/processed/final_dataset.csv` | 높음 |
| 4 | 데이터 확인 화면 구현 | 원천/가공 데이터 테이블 및 차트 | `pages/1_data_view.py` | 중간 |
| 5 | 거시 변수 탐색 화면 구현 | 변수 그룹별 추이와 상관관계 확인 | `pages/2_macro_view.py` | 중간 |
| 6 | SARIMAX 모델 구현 | 학습, 예측, 백테스트 함수 | `utils/model_sarimax.py` | 높음 |
| 7 | 예측 화면 구현 | 예측 차트, 평가 지표, 방향성 정확도 | `pages/3_forecast.py` | 높음 |
| 8 | 노트북 실험 | 데이터/모델 실험 기록 | `notebooks/test.ipynb` | 낮음 |
| 9 | README 정리 | 팀 공유용 설치/실행/데이터 설명 | `README.md` | 중간 |

## 개발 순서

1. `requirements.txt` 설치 후 `streamlit run app.py`로 앱 실행을 확인합니다.
2. `data/raw/`에 수집 데이터를 저장하고, `utils/preprocessing.py`로 최종 데이터셋을 만듭니다.
3. `pages/1_data_view.py`, `pages/2_macro_view.py`에서 데이터가 정상 표시되는지 확인합니다.
4. `utils/model_sarimax.py`로 SARIMAX 백테스트를 구현합니다.
5. `pages/3_forecast.py`에서 예측 결과와 RMSE, MAPE, 방향성 정확도를 표시합니다.
